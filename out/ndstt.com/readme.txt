# Installation instructions for YSMenu on the DSTT

1. Download YSMenu from here: https://gbatemp.net/download/35737/
2. Extract the YSMenu .7z file.
3. Copy the files inside of the DSTTi-Clone YSMenu folder to the root of your microSD card.

The original TTMenu will not be archived on this site due to it containing code which intends to brick clones of the DSTT.

Cheats: TTMenu/usrcheat.dat
