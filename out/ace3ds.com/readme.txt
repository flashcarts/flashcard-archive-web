# Installation instructions for YSMenu on the Original Ace3DS

Note that YSMenu will not work on an Ace3DS+ nor an Ace3DS X. Use the kernel in the `Ace3DS+_R4iLS` folder if you have one of those carts.

1. Download YSMenu from here: https://gbatemp.net/download/35737/
2. Extract the YSMenu .7z file.
3. Copy the files inside of the DSTTi-Clone YSMenu folder to the root of your microSD card.
4. Make a copy of `TTMenu.dat`, then rename this copy to `3DSCARD.DAT`.

Cheats:
    - YSMenu: TTMenu/usrcheat.dat
    - Original Ace3DS Kernel (old): SYSTEM/usrcheat.dat
