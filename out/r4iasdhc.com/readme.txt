# Installation instructions for YSMenu on r4iasdhc.com carts

Please follow the instructions to install YSMenu here: https://flashcarts.net/ysmenu-compat-ext.html

If your flashcard isn't listed on the website above, please open an issue on this repository!

Cheats:
    - YSMenu: TTMenu/usrcheat.dat
    - r4iasdhc.com Kernel 1.4.1H (old): SYSTEM/usrcheat.dat
