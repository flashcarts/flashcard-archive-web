# Installation instructions for YSMenu on r4i-gold.com flashcards

Warning:
    Only a few r4i-gold.com flashcarts are supported by YSMenu. Running YSMenu on an unsupported cart will cause it to brick!
    Supported carts:
        - r4i-gold.com V1.4.1-V1.4.4
        - r4i-gold.com Gold 3DS RTS
    If you have one of the following carts, YSMenu will BRICK them so do not attempt to use it:
        - r4i-gold.com Gold 3DS PRO 
        - Any r4i-gold.com cart with a year number

r4i-gold.com Gold 3DS RTS:
    - Make sure that the label mentions RTS, otherwise this can brick your flashcart!
    1. Download YSMenu from here: https://gbatemp.net/download/35737/
    2. Extract the YSMenu .7z file.
    3. Copy the files from the `DSTTi-Clone YSMenu` folder to the root of your microSD card.

r4i-gold.com V1.4.1-V1.4.4:
    1. Download YSMenu from here: https://gbatemp.net/download/35737/
    2. Extract the YSMenu .7z file.
    3. Copy the files from the `DSTTi-Clone YSMenu` folder to the root of your microSD card.
    4. Download R4.dat from the `/YSMenu/DEMON_common/` folder and place it onto the root of your microSD card.

Cheats:
    - YSMenu: TTMenu/usrcheat.dat
    - BL2CK: __rpg/cheats/usrcheat.dat
    - r4i-gold.com Gold 3DS RTS Kernel (old): SYSTEM/cheats/usrcheat.dat
    - r4i-gold.com V1.4.1-1.4.4 Kernel (old): R4iMenu/usrcheat.dat
    - r4i-gold.com Gold Pro/2013 Wood Kernel (old): __rpg/cheats/usrcheat.dat
