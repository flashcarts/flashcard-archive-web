# Installation instructions for YSMenu on ndsill.net flashcards

Note that ndsill.net's 3DS flashcards are not supported by YSMenu.

1. Download YSMenu from here: https://gbatemp.net/download/35737/
2. Extract the YSMenu .7z file.
3. Copy the files inside of the DSTTi-Clone YSMenu folder to the root of your microSD card.
4. Make a copy of `TTMenu.dat`, then rename this copy to `iLL.iL`

Cheats;
    - YSMenu: TTMenu/usrcheat.dat
    - 3DS Kernel 2.60: SYSTEM/usrcheat.dat
    - Original Kernel 1.45 (old): SYSTEM/usrcheat.dat
