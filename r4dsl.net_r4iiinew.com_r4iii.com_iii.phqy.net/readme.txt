# Installation instructions for YSMenu on r4dsl.net/r4iiinew.com/r4iii.com/iii.phqy.net flashcards

1. Install the stock kernel for your flashcard.
2. Download YSMenu from here: https://gbatemp.net/download/35737/
3. Extract the YSMenu .7z file.
4. Copy the files from the `R4-Clone YSMenu` folder to the root of your microSD card.

Cheats:
    - YSMenu: TTMenu/usrcheat.dat
    - r4dsl.net Kernel 3.13: _system_/CHEAT.dat
    - r4dsl.net Kernel 4.03: R4iMenu/usrcheat.dat
