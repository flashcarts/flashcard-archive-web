# Installation instructions for YSMenu on r4i3d.com carts

DO NOT install YSMenu on any r4i3d.com cart with a year number or 'new' written on the label. Doing so will brick your flashcard!

1. Download YSMenu from here: https://gbatemp.net/download/35737/
2. Extract the YSMenu .7z file.
3. Copy the files inside of the `DSTTi-clone YSMenu` folder to the root of your microSD card.
4. Make a copy of `TTMenu.dat`, then rename this copy to `R4i.3DS`.

cheats:
    - YSMenu: TTMenu/usrcheat.dat
    - BL2CK: __rpg/cheats/usrcheat.dat
    - r4i3d.com New/2013-2015 Kernels (old): __rpg/cheats/usrcheat.dat
