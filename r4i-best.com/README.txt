# Installation instructions for r4i-best.com's R4i-SDHC flashcard

1. Download YSMenu from here: https://gbatemp.net/download/35737/
2. Extract the YSMenu .7z file.
3. Copy the files inside of the `R4i-SDHC YSMenu` folder to the root of your microSD card.

Cheats:
    - YSMenu: TTMenu/usrcheat.dat
    - Best / Gold Kernel 2.02: SYSTEM/cheats/usrcheat.dat
    - Green Kernel 1.4: _system_/cheat.dat
    - R4i-SDHC DEMON Kernel 1.08b (old): R4iMenu/usrcheat.dat
